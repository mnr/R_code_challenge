Create a custom door combination that looks like a date

A friend has a combination lock on their rental apartment. They're able to set the combination over the internet and then send that combination to a guest, but they need a way to generate custom door codes. Their lock uses six digits and each digit must be unique. 
 
Your task: write a program to create a combination that's easy to remember, but hard to guess, like a calendar date.

Parameters

None

Result

int: A six-digit number that could be a valid date in YYMMDD format

Constraints

  • It should look like a date in YYMMDD format
  • It should be six digits long
  • All digits in the code should be unique
  • A valid example is: 591230. This would be December 30, 1959
  • An Invalid example is 010101. “0” and “1” repeat.
  • An invalid example is 691345. There is no thirteenth month, nor a 45th day of the month
