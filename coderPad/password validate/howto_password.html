Create a Password Validator

If you or a friend are using insecure passwords, you're placing your home and business in jeopardy. So it's really important to help everyone choose good passwords. 
 
For this challenge, we'll define a secure password in this way;
  • more than 10 characters - length
  • at least one lower case letter - lowercase
  • at least one uppercase letter - uppercase
  • at least one number - number
  • and at least one punctuation character - punctuation
 
Your task: Return a named vector indicating which tests a password passes or fails.

Parameters

aPassword: A string containing a possible password

Result

passwordTestResults: A named vector with a logical TRUE or FALSE value showing the result of each password test

Example 1:

Input: “abc123”
Result: length   lowercase   uppercase      number punctuation 
        FALSE    TRUE        FALSE         TRUE    FALSE

Example 2:

Input: “aB3- cat horse”
Result: length   lowercase   uppercase      number punctuation 
       TRUE      TRUE       TRUE            TRUE   TRUE
