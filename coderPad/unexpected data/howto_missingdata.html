Recognize different types of elements

You’ll be given a list of unknown values.
 
Your task: Return a value according to the following rules:
  • The sum of any numbers in the list, each rounded up to the nearest integer...
  • plus… 1 if TRUE, 0 if FALSE ...
  • plus...The average length of all strings …
  • minus...The count of any empty strings.
  • Ignore anything else, including (but not limited to) NA

Parameters

mysteryMeat: A list full of unexpected elements

Result

return: A number based on the above rules

Example 1:

Input: mysteryMeat <- list("hello", 1, 3.14, TRUE, "")
Result: 10
Explanation:
  • sum of any numbers, each number rounded up = 1 + 4 = 5
  • TRUE = 1, FALSE = 0 = 1
  • average length of strings = 5
  • count of empty strings = 1
  • 5 + 1 + 5 - 1
 
Example 2:
Input: mysteryMeat <- list(3, “mice”, “ran”, 2, FALSE, “alabama")
Result: 9.666667
Explanation:
  • sum of any numbers, each number rounded up = 3 + 2 = 5
  • TRUE = 1, FALSE = 0 = 0
  • average length of strings = 4.66667
  • count of empty strings = 0
  • 5 + 0 + 4.66667 + 0 = 9.66667
