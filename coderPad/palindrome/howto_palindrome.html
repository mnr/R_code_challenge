Check if a Word is a Palindrome

Palindromes are a word or phrase that reads the same backward as forward. For example: race car or madam are both palindrome. 
 
Your task: Given a word, return TRUE if the word is a palindrome, FALSE if not.

Parameters

testThisWord: A string to test if it is a palindrome

Result

Logical: return TRUE if the word is a palindrome, FALSE if not.

Example 1:

Input: race car
Result: TRUE

Example 2:

Input: watermelon
Result: FALSE
